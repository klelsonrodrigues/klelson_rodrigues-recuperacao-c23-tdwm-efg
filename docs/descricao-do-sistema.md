Descrição do Sistema
1. Nome do Site

TechFuture Store.
2. Objetivo

O sistema tem como objetivo oferecer uma plataforma de vendas online onde usuários podem navegar por um catálogo de produtos tecnológicos, gerenciar um carrinho de compras e realizar pedidos de forma segura e intuitiva.
3. Público-alvo

Consumidores entusiastas de tecnologia, estudantes e profissionais que buscam dispositivos eletrônicos e acessórios com facilidade de compra via web.
4. Funcionalidades Principais

As funcionalidades que compõem o escopo de uso do sistema são:

    Cadastro de Usuário: Permite que novos clientes criem uma conta fornecendo dados pessoais.

    Login de Usuário: Sistema de autenticação para acesso à área restrita e histórico de pedidos.

    Busca de Produtos: Barra de pesquisa para localização rápida de itens por nome ou categoria.

    Navegação por Categorias: Menu lateral e superior para explorar diferentes seções do site.

    Formulário de Contato: Espaço para o cliente enviar dúvidas ou solicitações ao suporte.
