Este repositório contém a documentação completa do projeto prático de testes de software para o componente curricular de Gestão de Configuração e Testes de Software.
 Objetivo

O objetivo desta atividade é aplicar na prática os conceitos de planejamento, elaboração, execução e registro de testes, utilizando o versionamento Git para organizar a documentação.
 Site Testado

    Nome: TechFuture Store.

    Tipo: E-commerce fictício.

    Funcionalidades Avaliadas: Cadastro, Login, Busca e Formulário de Contato.

Estrutura do Repositório

De acordo com os requisitos da atividade, o repositório está organizado da seguinte forma:

    docs/:

        descricao-do-sistema.md: Informações gerais sobre o site escolhido.

    tests/:

        plano-de-testes.md: Estratégia e escopo dos testes.

        casos-de-teste.md: Detalhamento dos passos e resultados esperados.

        execucao-dos-testes.md: Registro real dos resultados e evidências.

        registro-de-defeitos.md: Detalhamento de falhas encontradas durante os testes.

        relatorio-final.md: Resumo consolidado e conclusão sobre a qualidade.

 Ferramentas Utilizadas

    Git/GitHub: Para controle de versão e publicação.

    Markdown: Para formatação de toda a documentação.
